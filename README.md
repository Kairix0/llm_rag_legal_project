# Legal RAG & LLM Experiments

Proyecto personal de **IA aplicada con LLMs**, enfocado en:
- experimentación con modelos de lenguaje
- pruebas comparativas entre distintos LLMs
- generación de datasets legales
- consultas sobre documentos jurídicos usando RAG

## Estructura del proyecto

- `src/llm_simple/`: pruebas simples con distintos modelos LLM
- `src/rag_experiments/`: experimentos de RAG y procesamiento de PDFs
- `src/datasets/`: datasets legales y documentos fuente
- `data/`: preguntas y resultados de evaluación en Excel
- `docs/`: notas técnicas y documentación de aprendizaje

## Tecnologías
- Python
- LLMs (OpenAI, Mistral, modelos open-source)
- Procesamiento de documentos (PDF)
- Embeddings y recuperación de contexto (RAG)

## Nota
Este proyecto utiliza herramientas y modelos open-source como dependencias.
El foco está en la **integración, experimentación y uso práctico de LLMs**.
